Help

To view details about a server, click/tap on it and the server's details will appear. You can search servers by address, IP address, owner, country, and game-style. You can search mutliple fields by seperating them with commas. If it is taking a long to to query the API it may be due to connection errors or many requests being handled at once, please wait a short time then refresh. If a server's information is incorrect, please report it to the BZList author The Noah (make sure to look at the update times).

If no servers appear online, it may be due to the API being down or a local error. Please contact me if this does happens (but it should not happen often if it all).
