Help

To view details about a server, click or tap on it and the server's details should appear, including current player count, teams and game mode.

To search for a specific server, use the search field above the server list to search for a specific server. Servers may be searched by player count, address, IP, public title, game-style and more. You can search mutliple fields by seperating them with commas.

If a server query is taking a long time, it may be due to connection errors or many requests being handled at once, please wait a short time before attempting another query.

If a server's statistics or information is incorrect, please report it to the BZList author The Noah.

If no servers appear online, it may be due to the main server list being down or a local error. (This should not happen often.)
